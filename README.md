## Refrigerator-Management-Application
# :meat_on_bone:내 손안의 냉장고
<br/>

## 서비스 개요
내 손안의 냉장고는 **효율적인 냉장고 관리를 위한 애플리케이션**입니다.
<br/>

### 핵심 기능
- 바코드와 유통기한 인식을 통해 제품을 쉽고 빠르게 등록할 수 있습니다.
- 알림을 통해 사용자가 제품의 남은 유통기한을 인지할 수 있습니다.
- 냉장고 내부의 재료로 가능한 음식 레시피를 볼 수 있습니다.
<br/>
